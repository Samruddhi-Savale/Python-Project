# Python-Programming-Project

A collection of beginner-to-intermediate Python projects demonstrating problem-solving, logic building, and basic application development skills.

---

## Project Overview

This repository contains multiple independent Python scripts that demonstrate:
- Logical problem-solving and algorithmic thinking
- Use of conditionals, loops, and functions
- Console-based user interaction

---

## Project Files
- **Task_1.py – Calculator**
  - Performs basic arithmetic operations such as addition, subtraction, multiplication, and division.
  - <a href = "https://github.com/Samruddhi-Savale/Python-Project/blob/main/Task_1.py"> Code </a>

- **Task_2.py – ATM Simulator**
  - Simulates basic ATM operations like balance check, deposit, and withdrawal using conditional logic.
  - <a href = "https://github.com/Samruddhi-Savale/Python-Project/blob/main/Task_2.py"> Code </a>

- **Task_3.py – Random Password Generator**
  - Generates a strong random password based on user-defined length and character types.
  - <a href = "https://github.com/Samruddhi-Savale/Python-Project/blob/main/Task_3.py"> Code </a>

- **Task_4.py – Rock, Paper, Scissors Game**
  - A console-based game where the user plays against the computer.
  - <a href = "https://github.com/Samruddhi-Savale/Python-Project/blob/main/Task_4.py"> Code </a>

*(Each script is independent and can be run separately.)*

---

## Requirements
- Python 3.10 or higher
- No external libraries required (uses standard Python libraries only)

---

## How to Run the Project
1. Clone the repository:
   git clone https://github.com/Samruddhi-Savale/Python-Project.git

2. Open the project folder.

3. Run any script using:
   python Task_1.py
   python Task_2.py
   python Task_3.py
   python Task_4.py
